# pet-care
PetCare is an android app that represents the medical booklet of your pet. Purpose of the app is to make it easier for its user to have access to medical information at any time.

# Uses
Inside the app user can register the pet and its personal information like name, age, breed etc. After that multiple categories are available for the user to keep tab for daily things such as medication, vaccinations, bath, grooming and most important organized action to register medical exams.
